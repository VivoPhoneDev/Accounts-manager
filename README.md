# 🎮 Game Account Manager

A small console application for managing game accounts.

## ✨ Features

- ➕ Add new accounts
- 📋 View all saved accounts
- 🔍 Search accounts by login
- 💾 Save account data to JSON
- 📂 Load saved data between program launches

## 🛠 Technologies

- 🐍 Python
- 📄 JSON

## 🚀 How to Run

1. Download or clone the repository.
2. Open the project folder in a terminal.
3. Run:

```bash
python run.py
